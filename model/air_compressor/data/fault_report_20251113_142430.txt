【故障类型 1】主机过载
================================================================================
检出次数: 43次 (62.32%)

► 关键故障点:
  1. 主电机输出电流异常升高
  2. 主电机输出功率超负荷运行
  3. 主电机总功耗差值增大
  4. 排气温度上升
  5. 主电机转速略有下降

► 可能原因:
  1. 负载超过设备额定功率
  2. 压缩机内部磨损
  3. 润滑系统异常
  4. 进气过滤器堵塞

► 处理建议:
  1. 确认实际运行压力和产气量是否超过压缩机铭牌额定值；如有变频器，检查变频器运行频率和电流限制设置。
  2. 进行定期的主机大修/检查，特别关注轴承、转子与壳体之间的间隙，判断是否存在抱死或严重摩擦。
  3. 确认润滑油位在视镜的正常范围内；取样分析润滑油的黏度、酸值和颗粒物含量，若油质劣化严重需立即更换。
  4. 检查并记录进气压差，若压差超过制造商规定的阈值（如0.05 bar），应立即清洁或更换滤芯。

【故障类型 104】排气温度高 - 预警
================================================================================
检出次数: 8次 (11.59%)

► 关键故障点:
  1. 排气温度开始上升
  2. 主电机输出电流轻微波动
  3. 主电机功率轻微增加
  4. 风机转速升高
  5. 供气压力轻微波动

► 可能原因:
  1. 冷却系统效率下降
  2. 环境温度过高
  3. 压缩比过大
  4. 润滑油冷却效果差

► 处理建议:
  1. 检查油冷却器和后冷却器是否堵塞，如为水冷型，检查水流量和水质是否符合要求，水温不应过高。
  2. 确保机房温度不超过40°C（或制造商指定限值），检查排风系统是否能有效排出热空气。
  3. 适当降低卸载压力设定值，以减小压缩比，降低主机排气温度。
  4. 检查油过滤器是否堵塞（测量压差），检查温控阀（油断路阀）是否动作正常，确保油能流经冷却器。
