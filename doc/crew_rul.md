# 轴承剩余寿命 (RUL) 预测系统

## 应用介绍

轴承剩余寿命预测系统是一个基于深度学习的工业设备智能维护解决方案。该系统能够通过分析轴承运行过程中的振动信号，实时评估轴承的健康状态并预测其剩余可用时间，帮助企业从“事后维修”转向“预防性维护”，降低非计划停机风险。

### 主要功能

- **多版本模型支持**：提供多个预训练的深度学习模型版本（如 v4100, v4200, v4400 等），满足不同精度需求。
- **自动化特征提取**：自动从原始振动信号中提取均方根（RMS）、偏度、峭度、波形因子等时域特征，以及平均频率、频率标准差等频域特征。
- **直观的可视化诊断**：通过半圆仪表盘、健康度百分比卡片和进度条对比图，清晰展示轴承的当前寿命状态。
- **智能维护建议**：根据预测结果自动判定轴承状态（健康、轻微退化、严重退化），并给出相应的专业维护建议。

## 模型介绍

该应用集成了先进的深度学习架构，结合了时序特征提取与注意力机制，以实现对轴承退化过程的高精度建模。

### 模型特点

- **时序注意力机制 (Temporal Attention)**：采用自定义的注意力层，能够自动聚焦于对寿命预测最关键的时序片段。
- **多维特征融合**：融合了水平和垂直两个方向的振动信号特征，提供更全面的状态评估。
- **滑窗分析**：采用固定长度（WINDOW_SIZE=25）的滑动窗口对连续样本进行预测，确保预测结果的稳定性和连续性。
- **鲁棒的归一化处理**：每个模型版本均配有专属的归一化参数（Scaler），确保不同工况下数据的处理一致性。

### 使用方法

1. **选择数据源**：在“数据源选择”下拉菜单中选择待分析的轴承数据集文件夹。
2. **选择模型版本**：从可用的模型列表中选择一个版本（推荐使用最新版本 v4400）。
3. **调整分析区间**：使用“数据起始索引”滑块选择想要分析的时间点。
4. **运行预测**：点击“🚀 运行寿命预测”按钮。
5. **结果解读**：
    - 查看**仪表盘**获取直观的 RUL 分钟数。
    - 阅读**预测结果面板**获取详细的健康状态描述和维护建议。
    - 参考**对比图**了解当前寿命在整个生命周期中的位置。

### 模型选择说明

应用扫描 `model/crew_rul/model` 目录下的 `.h5` 文件。不同版本的模型在训练数据量和超参数上有所差异，用户可根据实际预测效果进行切换。

| 模型版本 | 核心技术 | 适用场景 |
|----------|----------|----------|
| v4400 (最新) | Attention + Deep CNN | 综合精度最高，推荐首选 |
| v4100 - v4300 | 多层感知器/基础CNN | 历史版本，用于对比分析 |

## 结果解释

预测完成后，系统会输出以下关键信息：

- **剩余使用寿命 (RUL)**：预测轴承还可以正常运行的具体分钟数。
- **健康度 (%)**：将预测 RUL 与设定的寿命上限（RUL_CAP=150）进行比例化展示。
- **健康状态判定**：
    - **健康 (绿色)**：轴承状态良好，建议继续正常运行。
    - **轻微退化 (黄色)**：出现初步磨损迹象，建议加强监控。
    - **严重退化 (红色)**：面临失效风险，强烈建议立即更换。
- **维护建议**：基于当前状态给出的专业化操作指引。
