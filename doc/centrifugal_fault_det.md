# 离心泵故障诊断应用

## 应用介绍

离心泵故障诊断应用是一个用于工业离心泵健康监测和故障分析的智能化工具。该应用通过分析离心泵运行过程中的流量、压力、温度以及控制阀开度等多维传感器数据，能够实时识别多种典型的设备故障，并提供详细的诊断建议。

### 主要功能

- **多维传感器数据分析**：整合分析出料流量、入口/出口压力、罐体压力及温度等关键运行参数。
- **智能化故障识别**：采用深度学习模型，自动识别离心泵的正常运行及五类典型故障状态。
- **详细故障分析报告**：自动生成包含故障统计、指标异常说明、根本原因分析及处理建议的完整报告。
- **可视化结果展示**：通过故障分布图和置信度分析图，直观展示诊断结果的可靠性。

## 模型介绍

该应用基于 PaddlePaddle 深度学习框架开发，核心采用了一维卷积神经网络（1D-CNN）架构，专门针对工业时间序列数据的特征提取进行了优化。

### 模型特点

- **卷积特征提取**：利用 1D-CNN 架构，能够从传感器数据的动态变化中捕捉细微的故障征兆。
- **高精度分类**：经过针对性训练，模型在正常状态、阀门卡涩、入口堵塞、汽蚀、气缚等状态上具有较高的识别精度。
- **端到端诊断**：实现了从原始数据输入到详细故障诊断建议输出的完整端到端流程。
- **专家经验集成**：将深度学习预测结果与离心泵运维专家知识库深度结合，提供具有实际指导意义的处理建议。

### 使用方法

1. **准备数据**：在应用界面上传包含传感器数据的 CSV 文件，或者从下拉菜单中选择预置的样例数据。
2. **加载分类器**：应用启动时会自动加载预训练模型。
3. **执行诊断**：点击“执行故障诊断”按钮，系统将对输入数据进行批量分析。
4. **查看分析结果**：
    - **诊断摘要**：查看数据源信息、样本总数及主要诊断结论。
    - **故障统计图表**：观察故障类型的检出次数分布及占比。
    - **置信度分布**：分析模型对各类故障预测的把握程度。
    - **详细故障分析报告**：阅读报告中的关键指标异常描述、可能原因及具体的维修处理建议。

### 模型选择说明

当前版本应用默认加载经过优化的 1D-CNN 故障诊断模型，其主要参数如下：

| 模型组件 | 技术细节 | 说明 |
|----------|----------|------|
| 核心架构 | 1D-CNN (FaultNet) | 包含多层卷积与池化，专门处理序列特征 |
| 训练框架 | PaddlePaddle | 国产领先的深度学习平台 |
| 输入特征 | 9维传感器数据 | 涵盖流量、压力、温度、阀位等 |
| 诊断类别 | 6类状态 | 1类正常 + 5类典型故障 |

## 结果解释

应用输出的诊断报告包含以下关键维度：

- **故障分布统计**：展示在分析样本周期内，各种状态出现的频率和平均置信度。
- **关键指标异常**：指出导致该诊断结论的具体物理指标异常（如“泵入口压力明显降低”）。
- **可能原因分析**：列出导致故障的潜在物理因素（如“过滤器堵塞”或“气源供应异常”）。
- **处理建议**：提供经过专家校验的维修保养步骤（如“立即检查泵入口过滤器，清理或更换滤芯”）。
