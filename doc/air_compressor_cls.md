# 空压机故障分类应用

## 应用介绍

空压机故障分类应用是一个专为工业空压机健康监测设计的工具。它能够通过分析电流、电压、功率、温度和压力等运行参数，实时识别和分类常见的空压机故障及其预警状态，为设备的预防性维护提供科学依据。

### 主要功能

- **多参数时域分析**：实时显示主电机电流、功率、排气温度、供气压力等关键指标的波形图。
- **智能故障识别**：基于深度学习与机器学习算法，精准识别主机过载、主机不平衡、风机故障等多种异常。
- **详细诊断报告**：提供故障点分析、根本原因溯源及针对性的维修处理建议。
- **数据源支持**：支持从 TDengine 数据库获取实时数据，或加载本地预置的模拟故障文件进行分析。

## 模型介绍

该应用集成了针对空压机运行特性优化的分类模型，通过对多维传感器数据的融合分析实现高精度的状态监测。

### 模型特点

- **自动特征提取**：能够从复杂的多维时间序列信号中自动提取反映设备健康状态的关键特征。
- **故障与预警双重监测**：不仅能识别已发生的故障类别，还能捕捉到早期的异常征兆并发出预警（Warning）。
- **专家知识库结合**：模型预测结果与内置的空压机专家知识库深度结合，提供具有可解释性的诊断建议。
- **全场景覆盖**：涵盖了主机、风机、电气系统等多方面的常见故障类型。

### 使用方法

1. **选择模型**：在应用界面选择所需的预训练模型（如 `fault_model_lightgbm`）。
2. **选择数据源**：从预置的测试文件列表中选择一个示例文件，或通过数据库连接获取实时数据。
3. **执行分析**：点击“处理数据”按钮，应用将加载数据并进行模型推理。
4. **查看结果**：
    - 观察**时域波形图**，识别参数异常趋势。
    - 阅读**故障诊断分析**部分，获取预测类别、故障点、原因及处理建议。

## 结果解释

模型分析完成后，应用会输出详尽的诊断结果：

- **时域波形图**：直观展示电流、功率、温度、压力等参数的变化，辅助人工校验。
- **预测类别**：明确指出当前设备处于正常状态还是具体的故障/预警状态。
- **故障点分析**：详细列出触发诊断的异常指标（如“主电机输出电流持续升高”）。
- **根本原因**：分析导致该故障的潜在物理原因（如“进气过滤器堵塞”）。
- **处理建议**：提供具体的检查、维修或保养方案（如“清洁或更换滤芯”）。
