# 轴承剩余寿命(RUL)预测解决方案

## 1. 概述
滚动轴承是旋转机械中最重要的零部件之一，其运行状态直接影响整机性能。轴承一旦损坏，可能导致非计划停机甚至重大事故。本方案通过实时分析轴承的振动信号，利用深度学习模型预测其剩余使用寿命（Remaining Useful Life, RUL），实现从“事后维修”到“预测性维护”的转变，延长设备寿命并降低维护成本。

## 2. 数据采集方案
轴承剩余寿命预测高度依赖高频振动原始信号。

### 2.1 硬件选型
| 设备名称 | 推荐型号 | 关键参数 | 备注 |
|---|---|---|---|
| **压电式加速度计** | PCB Piezotronics 352C33 | 灵敏度: 100 mV/g, 频率范围: 0.5-10kHz | 用于捕捉高频微弱振动 |
| **数据采集卡** | NI USB-4431 | 采样率: 102.4 kS/s, 24位分辨率 | 支持高频同步采样 |
| **安装底座** | 磁力吸附座 / 螺纹安装 | 刚性连接 | 确保信号传输不失真 |

### 2.2 采集点位与参数
- **采集点位**：
    - **水平方向**：安装于轴承座水平中心线位置。
    - **垂直方向**：安装于轴承座正上方位置。
- **采集参数**：
    - **采样频率**：25,600 Hz
    - **触发方式**：连续触发，每秒保存一个数据文件。
    - **信号调理**：内置 IEPE 供电，信号经过抗混叠滤波处理。

## 3. 方案架构
本方案采用时序深度学习预测架构：
- **感知层**：传感器以高采样率（如 25.6kHz）采集轴承的水平和垂直振动信号。
- **预处理层**：进行带通滤波和多维时域、频域特征提取。
- **模型层**：基于带有**时序注意力机制（Temporal Attention）**的深度神经网络，对历史退化趋势进行建模。
- **预测层**：输出当前时刻距离失效的剩余时间（分钟）。

## 3. 数据准备
### 3.1 原始信号
- **采样频率**：25600 Hz
- **信号通道**：水平振动 (Horizontal)、垂直振动 (Vertical)
- **数据格式**：每秒一个 CSV 文件，记录 25600 个采样点。

### 3.2 特征工程
提取 18 维复合特征（水平/垂直各 9 维）：
- **时域特征**：有效值 (RMS)、偏度、峰度、峰峰值、波形因子、脉冲因子等。
- **频域特征**：重心频率、频率标准差。
- **数据处理**：使用 Butterworth 带通滤波器去除噪声。

## 4. 模型开发
### 4.1 算法模型
- **核心架构**：深度时序模型 + 时序注意力机制。
- **注意力机制**：自动识别不同时间步对预测结果的重要性，增强模型对退化转折点的敏感度。
- **训练目标**：最小化对数转换后的 RUL 预测误差。

### 4.2 性能优化
- **滑动窗口**：使用 25 个时间步的窗口进行连续预测。
- **标准化**：使用 Scaler 对输入特征进行动态缩放。

## 5. 部署与可视化
- **仪表盘展示**：直观展示剩余寿命数值、健康百分比。
- **状态评估**：
    - **健康**：RUL > 75 min
    - **轻微退化**：37.5 < RUL <= 75 min
    - **严重退化**：RUL <= 37.5 min
- **趋势分析**：通过对比图展示当前 RUL 在全生命周期中的位置，并给出针对性的维护建议。
