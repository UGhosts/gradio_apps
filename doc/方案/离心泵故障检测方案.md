# 离心泵故障种类识别解决方案

## 1. 概述
在工业生产中，离心泵是核心的流体输送设备。当离心泵发生故障（如流量波动、输送能力下降）时，传统的现场检查方法费时费力。本方案通过AI模型实时监测离心泵的运行参数（流量、压力、液位、温度等），自动识别多种故障类型，帮助操作员快速定位问题，提高处理效率，降低生产风险。

## 2. 数据采集方案
为了实现对离心泵运行状态的精准监控，本方案采用多参数融合采集策略。采集点位覆盖了泵组的动力源、进出口流体状态及控制执行机构。

## 3. 方案架构
本方案采用基于深度学习的分类预测架构：
- **数据层**：实时采集离心泵的进出口流量、压力、液位、温度及阀门开度等12个核心参数。
- **特征工程**：对原始时序数据进行归一化处理和特征提取。
- **模型层**：采用一维卷积神经网络（1D CNN）提取时序特征并进行多分类识别。
- **展示层**：通过可视化图表展示实时状态及故障诊断结果。

### 3.2 数据集说明
数据集包含正常状态及5种典型故障状态：
1. **正常状态** (Label 0)
2. **流量输送阀卡涩** (Label 1)
3. **入口堵塞** (Label 2)
4. **入口温度升高汽蚀** (Label 3)
5. **气缚故障** (Label 4)
6. **吸入罐压力控制阀卡涩** (Label 5)

## 4. 模型开发
### 4.1 数据预处理
- **归一化**：使用均值和标准差对各特征列进行归一化处理。
- **独热编码**：对故障类别标签进行One-hot编码。
- **形状重塑**：将数据重塑为 `[样本数, 1, 特征数]` 以适配1D CNN输入。

### 4.2 模型结构
采用LeNet风格的1D CNN结构：
- **Conv1D层**：提取局部时序特征。
- **ReLU激活**：增加非线性表达。
- **MaxPool1D层**：下采样，提取主要特征。
- **全连接层**：将提取的特征映射到故障类别空间。

## 5. 部署与可视化
- **实时监测**：系统实时接收传感器数据，经过模型推理给出当前状态。
- **故障告警**：当检测到异常状态时，系统自动识别故障种类并发出告警。
- **评估反馈**：通过混淆矩阵和评估数据对模型性能进行持续监控。
